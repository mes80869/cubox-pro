## AI 功能到底好不好用？

Cubox 的 AI 功能不是那种贴牌 AI，是真正集成到阅读流程里的。

**自动解读**：文章收藏进来就自动生成"摘要 + 核心观点"，不是简单的内容复述，而是帮你判断这篇文章到底在说什么、有哪些关键结论值得记录。对于那种 5000 字学术文章或行业分析报告，这个功能能省掉大量预读时间。

**幻影高亮**：相当于让 AI 帮你先读一遍，把重要的句子标出来。读长文的时候不用自己扫视全篇找重点，直接看 AI 标记的部分就能抓住主干。

**AI 问答**：对着收藏的文章提问，比如"这篇文章的核心论点是什么"、"帮我写一段分享这篇文章的话"，AI 会基于文章内容给出回答，而不是胡乱生成内容。
